# discord-bot
- A discord chat bot based on discord.js
- Written in TypeScript

# Installation
1. Get node.js from https://nodejs.org/en/download/
2. Clone the project
3. Install all required packages with command `npm  install`

# Usage
1. Create a new application and bot user in https://discordapp.com/developers/applications/me
2. Copy auth.json.example to auth.json and fill in the user authorization token
3. Add the bot to your server from URL 
`https://discordapp.com/oauth2/authorize?&client_id=<BOT_CLIENT_ID>&scope=bot&permissions=0`
4. Start the bot with command `npm start`
